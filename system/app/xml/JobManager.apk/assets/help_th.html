เพื่อ​ยืด​อายุ​การ​ใช้​งาน​แบตเตอรี่

อายุ​การ​ใช้​งาน​แบตเตอรี่​จะ​สัมพันธ์​กับ​การ​ใช้ CPU ​ปิด​แอ​พพลิ​เค​ชั่น​ทั้ง​หมด​ที่​ไม่​จำ​เป็น บาง​แอ​พพลิ​เค​ชั่น​อาจ​ไม่​แสดงบน​รายการ​โปรแกรม​ที่​กำลัง​ทำ​งานอยู่

1. บาง​แอ​พพลิ​เค​ชั่น​ที่​ทำ​งาน​ใน​พื้น​หลัง​อาจ​ทำ​ให้​สิ้น​เปลือง​กำลัง​ไฟ​แบตเตอรี่​โดย​ไม่​จำ​เ็ป็น​ หาก​ต้อง​การ​ปิด​แอ​พพลิ​เค​ชั่น ให้​ใช้​ปุ่ม​กลับ​แทน​ปุ่ม​หน้า​หลัก (​ปุ่ม​กลับ​จะ​ปิด​แอ​พพลิ​เค​ชั่น​ส่วน​ใหญ่ ขณะ​ที่​ปุ่ม​หน้า​หลัก​จะ​นำ​แอ​พพลิ​เค​ชั่น​มา​ไว้​ใน​โหมด​พร้อม​ใช้​งาน)

2. คุณ​สามารถ​แสดง​จำนวน​แอ​พพลิ​เค​ชั่น​ที่​ทำ​งาน​ใน​ปัจจุบัน​ร่วม​กับ Widget ​การ​ตรวจ​สอบ​โปรแกรม (​บาง​แอ​พพลิ​เค​ชั่น​อาจ​ไม่​แสดง) ​ควร​ใช้​ความ​ระมัด​ระ​วัง​เมื่ือติ​ด​ตั้ง​แอ​พพลิ​เค​ชั่น​ที่​ไม่​รู้จัก​หรือ​ไม่​ปลอดภัย เพราะ​อาจ​ทำ​ให้​สิ้น​เปลือง​กำลัง​ไฟ​แบตเตอรี่​มาก​เกิน​ไป

3. หลัง​จาก​ตั้ง​ค่า​บัญชี​ผู้​ใช้ Google ​แล้ว การ​สื่อ​สาร​ข้อมูล​จะ​เกิด​ขึ้น​ที่​พื้น​หลัง หาก​ต้อง​การ​ขยาย​เวลา​พร้อม​ใช้​งาน ให้​เลิก​ใช้​งาน​ซิงค์​อัตโนมัติ​ใต้​การ​ตั้ง​ค่า > ​บัญชี​ผู้​ใช้​และ​ซิงค์ และ​ซิ​งโครไนซ์​ด้วย​ตน​เอง​เมื่อ​จำ​เป็น

4. การ​เิ​ปิด Wi-Fi ​และบลูทูธ​ไว้​ตลอด​เวลา ​มัก​เป็น​สาเหตุ​หลัก​ที่​ทำ​ให้​แบตเตอรี่​หมด

5. ใน​กรณี​ที่​สิ้น​เปลือง​แบตเตอรี่​มาก​เกิน​ไป​หรือ​โทรศัพท์​ขัด​ข้ 8 ให้​เปิด​โทรศัพท์​ขึ้น​มา​ใหม่​โดย​กด​ที่​ปุ่ม​เปิด/​ปิด​พร้อม​กัน​นาน %d ​วินาที หรือ​ด้วย​การ​กด​ปุ่ม​เปิด/​ปิด​ค้าง​ไว้​และ​เลือก​ปิด​เครื่อง

เกี่ยว​กับ​ตัว​จัดการ RAM

​เพื่อ​ให้​กลับ​มา​เปิด​ใช้​งาน​ใหม่​ได้​อย่าง​รวด​เร็ว แอ​พพลิ​เค​ชั่น​ของ​ระบบ​ปฏิบัติ​การ Android ​จะ​คง​อยู่​ใน​หน่วย​ความ​จำ​แม้​ใน​ขณะ​ที่​แอ​พพลิ​เค​ชั่น​เหล่า​นั้น​ปิด​อยู่ เมื่อ​หน่วย​ความ​จำ​ไม่​เพียง​พอ ตัว​กำจัด​หน่วย​ความ​จำ​ต่ำ (LMK) ​จะ​ล้าง​หน่วย​ความ​จำ​โดย​อัตโนมัติ​โดย​ใช้​รายการ​การ​ใช้​งาน​ล่า​สุด​น้อย​ที่​สุด (LRU) ​หาก​คุณ​ล้าง​หน่วย​ความ​จำ​ด้วย​ตน​เอง อาจ​จะ​ใช้​เวลา​นาน​ใน​การ​เริ่ม​ใช้​แอ​พพลิ​เค​ชั่น.
