Förlänga batteritiden

Batteritiden beror på CPU-användningen. Avsluta onödiga program. Vissa program visas kanske inte i listan med program som körs

1. Vissa program som körs i bakgrunden kan påverka batteritiden negativt. Om du vill stänga ett program använder du tillbakaknappen i stället för hemknappen. (Tillbakaknappen stänger de flesta program, medan hemknappen sätter programmen i viloläge.)

2. Du kan visa antalet program som körs med widgeten för att övervaka program. (Vissa program visas kanske inte.) Var försiktig när du installerar okända eller osäkra program eftersom det kan påverka batteriförbrukningen

3. När du har skapat ett Google-konto sker datakommunikationen i bakgrunden. Om du vill utöka standby-tiden inaktiverar du automatisk synkronisering under Inställningar > Konton och synkning, och synkroniserar manuellt vid behov

4. Hög batteriförbrukning beror ofta på att Wi-Fi och Bluetooth är på

5. Vid problem med telefonen eller hög batteriförbrukning kan du starta om telefonen genom att trycka på strömknappen i 8 sekunder, eller genom att hålla in strömknappen och välja Stänga av

Om RAM-hanterare

För att aktivera snabb omstart behålls Android OS-program i minnet trots att de är avstängda. När det inte finns tillräckligt med minne kan LMK (low memory killer) rensa minnet automatiskt utifrån listan LRU (least recently used). Om du rensar minnet manuellt kan det ta längre tid att starta programmen.
